# 🥦 Vegetable Delivery App

## 📌 Project Overview
This is a backend application for a vegetable delivery system built using Spring Boot. 
It allows users to browse products, add items to cart, and place orders.

## 🚀 Features
- User Authentication (JWT)
- Product Management
- Cart System
- Order Placement
- Admin Panel (Upcoming)

## 🛠 Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT
- MySQL
- Maven

## 📂 API Endpoints
| Method | Endpoint | Description |
|--------|---------|------------|
| POST | /auth/login | User login |
| GET | /products | Get all products |
| POST | /orders | Place order |

## ⚙️ Setup Instructions
1. Clone repo
2. Configure database
3. Run using IntelliJ
